    Struct Lexer

    Lexer struct is a really customizable lexer.

    struct Lexer(T);

    Constructors

    NameDescription
    this (src, sp, ds, tz) This is the more general form of the previous constructor.
    this (src, tz) This constructor uses default separator and discarding_separator associative arrays.

    Methods

    NameDescription
    next_symbol () Returns next symbol, advances cursor. Returns Epsilon (0) on file end.

    Inner structs

    NameDescription
    Token Internal token type for packaging semantic information with the token

    Parameters

    NameDescription
    T Token enumeration type that is convertable to integer. There must be a epsilon token that has the 0 value.

    Example

    ---d enum Tokens { Epsilon = 0 Identifier, OpenParen, CloseParen, } Tokens tokenizer(string lex) { if (lex == "(") return Tokens.OpenParen; else if (lex == ")") return Tokens.CloseParen; return Tokens.Identifier; } string source_code = "abc ((def) ghi)"; // We use an alias for conciseness. alias MyLexer = Lexer!Tokens; auto lexer = Lexer!Tokens(source_code, &tokenizer); MyLexer.Token tk = lexer.next_symbol(); while (tk.type != Tokens.Epsilon) { // Do what ever you want to the token tk = lexer.next_symbol(); }
