    Function Parser.initFollow

    Initializes 'Follow' sets of each nonterminal.

    void initFollow(
      N starter
    );

    Follow set of a nonterminal contains every token that can appear right after that nonterminal ends.

    Example

    For this grammer. A = xBy | zBq | uBw B = a | b // definition of B doesn't matter in this case Follow(A) = {y, q, w}
