    Struct Parser

    Parser template creates a LL(1) parser for the given grammar and tokens.

    struct Parser(T, N);

    Constructors

    NameDescription
    this (rules, start) Constructs the parser structure.

    Methods

    NameDescription
    buildParsingTable () Builds LL(1) parsing table using first and follow sets.
    initFirst () Initializers "First" sets of each nonterminal.
    initFollow (starter) Initializes 'Follow' sets of each nonterminal.
    parse (token_list) Parses the given token_list.

    Parameters

    NameDescription
